HOW EXPORTING WORKS

A design is a folder, not a file. A lone .html renders unstyled once it's separated from its CSS, fonts and images. So this app only imports prepared folders — and the prompt below gets Claude to make one for you.

  1. Copy the prompt below.
  2. Paste it to Claude (the chat where you made your design), together with your design. Claude returns a self-contained folder — HTML + CSS + self-hosted fonts + local images — as a .zip.
  3. Unzip it, then import the FOLDER here (the “Choose Folder” tile) — not the .html inside.
  4. Select → Scale → Export.
